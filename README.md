Профессиональный сканер реестра Windows BAM (Background Activity Monitor) для выявления следов читов и подозрительных программ.

## Запуск

Требует **права Администратора**.

```cmd
BamDelKeys.exe
```

## Возможности

- Автоматическое сканирование всех путей BAM (Win10/Win11)
- Конвертация внутренних путей `\Device\HarddiskVolume` в читаемый формат `C:\`
- Детект удалённых файлов (следы)
- Проверка цифровых подписей
- Выявление запусков с внешних носителей (ghost devices)
- Детект подмены путей (blank/invisible символы)
- Сортировка по времени запуска (свежие — сверху)

## Таблица результатов

| Колонка | Описание |
|---|---|
| # | Порядковый номер |
| Executable Path | Полный путь к файлу (конвертированный) |
| SID | Security Identifier пользователя |
| Last Run (UTC) | Время последнего запуска |
| Status | `EXIST` / `DELETED` / `GHOST` |
| Trust | Статус цифровой подписи |
| Verdict | Итоговый вердикт |

## Вердикты

| Вердикт | Значение | Опасность |
|---|---|:---:|
| `[CLEAN]` | Чистый файл | ✅ |
| `[SUSPECT]` | Подозрительное имя | ⚠️ |
| `⚠ TRACE DETECTED` | Подозрительный файл удалён — след остался | 🔴 |
| `⚠ GHOST DEVICE` | Запуск с отключённого носителя | 🔴 |
| `⚠ GHOST+SUSPECT` | Отключённый носитель + подозрительное имя | 🔴🔴 |
| `⚠ BLANK PATH` | Путь пустой/невидимые символы — подмена | 🔴🔴 |
| `⚠ HIDDEN` | Ключ реестра скрыт / доступ запрещён | 🔴🔴 |
