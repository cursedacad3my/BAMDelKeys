# BAMDelKeys
Данная програма сканирует реестр Windows BAM (Background Activity Monitor) и выводит список всех запускавшихся `.exe` файлов с анализом на читы.

## Запуск

Требует **права Администратора**.

```
BAMDelKeys

```

## Что показывает таблица

| Колонка | Описание |
|---|---|
| Executable Path | Путь к запущенному файлу |
| SID | Пользователь (Security Identifier) |
| Last Run (UTC) | Время последнего запуска |
| Status | `EXIST` — файл есть / `DELETED` — удалён / `GHOST` — запуск с флешки |
| Trust | Цифровая подпись: `Signed` / `Unsigned` |
| Verdict | Итоговый вердикт (см. ниже) |

## Вердикты

| Вердикт | Значение |
|---|---|
| `[CLEAN]` | Чисто |
| `[SUSPECT]` | Путь содержит подозрительное слово |
| `⚠ TRACE DETECTED` | Подозрительный файл был удалён — след остался |
| `⚠ GHOST DEVICE` | Запуск с внешнего носителя (`\Device\HarddiskVolume...`) |
| `⚠ GHOST+SUSPECT` | Внешний носитель + подозрительное имя |
| `⚠ BLANK PATH` | Путь пустой или из невидимых символов — подмена |
| `⚠ HIDDEN` | Ключ реестра скрыт / доступ запрещён |
